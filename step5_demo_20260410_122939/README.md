# Pet 3D Localization Visualization Demo

## 项目概览

这是一个完整的宠物3D定位可视化系统演示，展示了从视频检测到3D空间定位的端到端流程。

## 技术架构

### 第1步：基础视频读取和YOLO检测
- 使用YOLOv8n模型检测视频中的猫和狗
- 输出像素坐标位置和置信度

### 第2步：Homography坐标转换系统
- 利用地面标记点标定的单应性矩阵
- 将像素坐标转换为物理世界坐标
- 验证标定精度（平均误差约7厘米）

### 第3步：RoomPlan 3D模型加载和处理
- 读取Apple RoomPlan生成的.usd文件
- 解析房间几何结构和家具布局
- 建立3D空间坐标系统

### 第4步：整合检测和3D定位
- 统一不同坐标系统
- 实现端到端的检测→定位流程
- 添加语义位置描述（如"房间中央"、"右下角"等）

### 第5步：3D可视化界面
- Web界面交互式可视化
- 支持帧级别的宠物位置显示
- 时间轴控制和统计分析

## 主要特性

### 🎯 高精度定位
- 100%的检测成功转换为3D坐标
- 平均定位精度达到厘米级别

### 📊 实时统计
- 宠物检测数量统计
- 空间分布分析
- 时间序列分析

### 🗺️ 交互式可视化
- 房间俯视图显示
- 帧级别的位置跟踪
- 轨迹热力图

### 🔍 多维度分析
- 像素检测 → 物理坐标 → 3D定位 → 语义位置
- 支持时间轴回放和跳转

## 应用场景

1. **宠物行为研究**: 分析宠物的活动模式和偏好区域
2. **智能家居**: 基于宠物位置的自动化控制
3. **安全监控**: 实时跟踪宠物位置，防止走失
4. **健康监测**: 监测宠物活动量和行为变化

## 技术优势

- **无需额外硬件**: 仅使用普通摄像头
- **高精度定位**: 结合计算机视觉和几何变换
- **实时处理**: 支持实时视频流分析
- **可扩展性**: 易于扩展到多房间、多摄像头场景
